midiToArray
===========

Tool which convert midi file in C array

Compilation
===========

cabal configure
cabal build

Dependency
==========
Tool need:
- Haskore
- honk > 1.2.0.0 (http://hackage.haskell.org/package/honk-1.2.0.0)

Known issues
============
Tool can play music in system speeker only from ghci interpreter with root.
